# 语音与AI分析说明（已整合）

该功能文档已整合至 `USAGE_GUIDE.md` 的“梦境记录页面”与“AI分析”章节中。

📌 请前往：`Read/USAGE_GUIDE.md` 查阅完整使用说明。

#### 功能特点
- **梦境详情播放**：在梦境详情页面播放原始语音
- **播放状态显示**：显示播放时长和状态
- **云存储支持**：语音文件自动上传到云存储

#### 使用方法
1. 在梦境详情页面查看梦境内容
2. 如果有语音记录，会显示"播放语音"按钮
3. 点击按钮播放原始录音
4. 播放过程中可以暂停

## 技术实现

### 语音录制
- 使用微信小程序 `wx.getRecorderManager()` API
- 支持MP3格式，16kHz采样率
- 最长录制60秒
- 自动权限检查和请求

### 语音识别
- 使用微信小程序语音识别API
- 自动将语音转换为文字
- 支持中文识别

### AI分析
- 集成Coze API进行梦境分析
- 支持情绪分析、标签提取、分类和解梦
- 多端点API调用机制，自动尝试不同版本
- API失败时自动使用模拟分析作为备用方案
- 完善的错误处理和重试机制

### 数据存储
- 语音文件上传到微信云存储
- 梦境数据保存到云数据库
- 支持语音文件下载和播放

## 注意事项

1. **网络要求**：AI分析需要网络连接
2. **权限要求**：首次使用需要授权录音权限
3. **文件大小**：语音文件会自动压缩以节省存储空间
4. **隐私保护**：所有数据都存储在用户的云存储中

## 故障排除

### 录音失败
- 检查是否已授权录音权限
- 确认设备麦克风正常工作
- 检查网络连接

### AI分析失败
- 检查网络连接
- 确认梦境内容不为空（至少3个字符）
- 对于短梦境，可以选择继续分析或添加更多描述
- 如果API服务不可用，系统会自动使用备用分析模式
- 查看错误提示信息

### 语音播放失败
- 检查网络连接
- 确认语音文件存在
- 尝试重新加载页面
