# 🚀 快速开始指南

## 5分钟快速配置

### 第1步：开通云开发（2分钟）

1. 打开微信开发者工具
2. 点击顶部 **"云开发"** 按钮
3. 点击 **"开通"** 按钮
4. 环境名称填写：`dream-wormhole`
5. 点击 **"创建"** 并等待完成

### 第2步：获取环境ID（1分钟）

1. 在云开发控制台，点击 **"设置"**
2. 复制 **"环境ID"**（类似：`dream-wormhole-xxxxx`）
3. 打开 `app.js` 文件
4. 替换第9行的环境ID：

```javascript
// app.js 第9行
env: '你的环境ID', // 粘贴刚才复制的环境ID
```

### 第3步：创建数据库集合（1分钟）

在云开发控制台 → 数据库：

1. 点击 **"添加集合"**
2. 创建集合 `dreams`（梦境记录）
3. 再次点击 **"添加集合"**
4. 创建集合 `planets`（星球数据）

### 第4步：测试运行（1分钟）

1. 在微信开发者工具中点击 **"编译"**
2. 在首页输入梦境内容：
   ```
   梦见自己在天空中自由飞翔，周围是彩色的云朵
   ```
3. 点击 **"✨ AI分析"** 按钮
4. 等待分析完成
5. 点击 **"保存到虫洞"**

### ✅ 完成！

现在你可以：
- ✅ 使用 Coze AI 分析梦境
- ✅ 自动分类到不同星球
- ✅ 在宇宙地图中查看
- ✅ 数据自动保存到云端

## ❓ 常见问题

### Q: AI 分析失败？
A: 检查网络连接，确保 Coze API 正常

### Q: 数据保存失败？
A: 确认云开发已开通，数据库集合已创建

### Q: 宇宙地图空白？
A: 先保存至少一条梦境数据
