## 🚀 使用步骤

### 第一步：开通微信云开发

1. 打开微信开发者工具
2. 点击顶部"云开发"按钮
3. 点击"开通"按钮
4. 填写环境名称：`dream-wormhole`
5. 等待创建完成（约1-2分钟）

### 第二步：获取云环境 ID

1. 进入云开发控制台
2. 点击"设置" → "环境设置"
3. 复制"环境ID"
4. 修改 `app.js` 中的环境ID：

```javascript
// app.js
wx.cloud.init({
  env: '你的环境ID', // 替换这里
  traceUser: true
});
```

### 第三步：创建数据库集合

在云开发控制台 → 数据库中创建以下集合：

#### 1. dreams 集合（梦境记录）
```
集合名称：dreams
权限设置：仅创建者可读写
```

#### 2. planets 集合（星球数据）
```
集合名称：planets
权限设置：仅创建者可读写
```

### 第四步：配置 Coze 智能体

#### 在 Coze 平台设置智能体提示词

访问 Coze 平台，编辑你的智能体，设置以下提示词：

```
你是一个专业的梦境分析师。请根据用户提供的梦境内容，分析其中的情绪和含义。

分析要求：
1. 识别梦境中的主要情绪（压力、快乐、焦虑、兴奋等），并给出0-100的数值
2. 提取3-5个关键词标签
3. 判断梦境类型（从以下类型中选择一个）：
   - 飞翔星球（包含飞翔、飞行、天空、自由等）
   - 海洋星球（包含海洋、水、游泳、潜水等）
   - 森林星球（包含森林、树木、自然、动物等）
   - 迷宫星球（包含迷宫、迷失、寻找、困惑等）
   - 城市星球（包含城市、建筑、街道、人群等）
   - 奇幻星球（包含魔法、超能力、怪物、奇幻生物等）
   - 恐怖星球（包含恐惧、追逐、黑暗、危险等）
   - 童年星球（包含童年回忆、学校、玩耍等）
4. 提供专业的梦境解析

请严格按照以下JSON格式返回，不要添加任何其他文字：
{
  "emotions": [
    {"name": "压力", "value": 数值},
    {"name": "快乐", "value": 数值},
    {"name": "焦虑", "value": 数值},
    {"name": "兴奋", "value": 数值}
  ],
  "tags": ["标签1", "标签2", "标签3"],
  "planetType": "星球类型名称",
  "interpretation": "详细的梦境解析内容"
}
```

## 🔧 常见问题

### Q1: AI 分析失败怎么办？

**原因**：
- Coze API 调用失败
- 网络问题
- API Key 或 Bot ID 错误

**解决方案**：
1. 检查网络连接
2. 确认 API Key 和 Bot ID 正确
3. 查看控制台错误信息
4. 检查 Coze 平台智能体是否正常工作

### Q2: 数据保存失败怎么办？

**原因**：
- 云开发未开通
- 数据库集合未创建
- 权限设置错误

**解决方案**：
1. 确认云开发已开通
2. 创建 dreams 和 planets 集合
3. 设置集合权限为"仅创建者可读写"
4. 修改 app.js 中的环境ID

### Q3: 宇宙地图显示空白？

**原因**：
- 没有保存过梦境
- 数据库查询失败

**解决方案**：
1. 先保存至少一条梦境
2. 检查控制台错误信息
3. 确认数据库中有数据

### Q4: Coze API 返回格式错误？

**原因**：
- 智能体提示词设置不当
- AI 返回了非 JSON 格式

**解决方案**：
1. 修改 Coze 智能体提示词
2. 强调返回格式必须是 JSON
3. 添加格式验证逻辑
