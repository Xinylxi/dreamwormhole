# 控制台警告修复指南

本文档说明如何修复小程序运行时出现的警告和性能问题。

## ✅ 已修复的问题

### 1. app.json 配置警告

**问题描述：**
```
⚠ 无效的 app.json rendererOptions.skyline["tagNameStyleIsolation"]
```

**原因：**
`tagNameStyleIsolation` 设置为 "legacy" 已废弃，应使用 "isolated"

**修复状态：** ✅ 已修复

**修复内容：**
已将 `app.json` 中的配置从 `"tagNameStyleIsolation": "legacy"` 更改为 `"tagNameStyleIsolation": "isolated"`

---

## ⚠️ 需要手动操作的问题

### 2. 数据库性能优化建议

**问题描述：**
```
⚠ 发起的如下数据库查询经自动测量疑似下问题，且可能较之高效的索引支持
```

**问题说明：**
控制台建议为 `dreams` 集合创建索引以优化查询性能。

**具体查询：**
```javascript
db.collection('dreams').where({
  _openid: 'o7i9-1_DpaM-Xr91puYxSZngYEp4'
})
.orderBy('createdAt', 'desc')
.get()
```

**推荐索引：**
- 字段 1: `_openid` (升序)
- 字段 2: `createdAt` (降序)

---

## 📋 操作步骤：创建数据库索引

### 方法一：使用控制台快速链接（最简单）

1. 在微信开发者工具的控制台中，找到如下绿色提示：
   ```
   索引建议：
   ```
   
2. 点击控制台提供的创建索引链接，自动跳转到云开发控制台

3. 确认索引配置后，点击保存

### 方法二：手动在云开发控制台创建

1. **打开云开发控制台**
   - 在微信开发者工具中，点击顶部菜单「云开发」
   - 或访问：https://console.cloud.tencent.com/tcb

2. **进入数据库管理**
   - 选择你的云开发环境
   - 点击左侧菜单「数据库」
   - 选择 `dreams` 集合

3. **创建索引**
   - 点击「索引管理」标签
   - 点击「添加索引」按钮
   - 添加第一个字段：
     * 字段名: `_openid`
     * 排序: `升序`
   - 添加第二个字段：
     * 字段名: `createdAt`
     * 排序: `降序`
   - 点击「确定」保存

4. **等待索引创建完成**
   - 索引创建可能需要几分钟
   - 创建完成后，查询性能将显著提升

## ❓ 常见问题

**Q: 索引创建失败怎么办？**
A: 确保：
- 云开发环境已正常开通
- `dreams` 集合已存在
- 字段名拼写正确（区分大小写）

**Q: 创建索引后还是有警告？**
A: 
1. 等待索引创建完成（可能需要几分钟）
2. 重启小程序
3. 清除缓存后重试

**Q: 索引会影响数据写入速度吗？**
A: 索引会略微增加写入时间，但大大提升查询速度，整体性能得到改善。
