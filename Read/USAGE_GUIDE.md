# 梦境虫洞 - 使用指南

## 🚀 快速开始

### 第一步：配置图标（重要！）

由于 tabBar 需要图标文件，请按以下步骤操作：

#### 方案一：添加图标文件
1. 在 `assets/icons/` 目录下添加以下图标：
   - `record.png` - 记录图标（81x81px）
   - `record-active.png` - 记录选中图标
   - `universe.png` - 宇宙图标
   - `universe-active.png` - 宇宙选中图标

2. 推荐图标来源：
   - [iconfont](https://www.iconfont.cn/) - 搜索"记录"、"宇宙"等关键词
   - [Feather Icons](https://feathericons.com/) - 简洁线性图标
   - [Material Icons](https://fonts.google.com/icons) - Google 图标库

#### 方案二：临时禁用 tabBar
如果暂时没有图标，可以在 `app.json` 中注释掉 `tabBar` 配置：

```json
{
  "pages": [...],
  "window": {...},
  // "tabBar": {...},  // 临时注释
  "style": "v2"
}
```

然后通过页面跳转访问宇宙地图。

## 📱 功能使用说明

### 1. 梦境记录页面（首页）

#### 记录新梦境
1. 在文本框中输入梦境内容（最多500字）
2. 点击"✨ AI分析"按钮
3. 等待AI分析情绪指数
4. 查看情绪标签（压力、快乐、焦虑、兴奋）
5. 点击"保存到虫洞"保存梦境

#### 查看最近梦境
- 在"最近梦境"区域可以看到最近的梦境记录
- 点击任意梦境可以查看详情
- 点击"查看宇宙地图 →"跳转到宇宙地图

#### 时空穿梭功能
- 点击"🚀 AI梦境引导"卡片
- 告诉AI你想做什么梦（功能开发中）

### 2. 宇宙地图页面

#### 浏览梦境星球
- 页面顶部显示宇宙地图可视化
- 不同颜色的星球代表不同类型的梦境
- 星球之间的线条是"虫洞"连接

#### 交互操作
- **点击星球**：选中星球，显示星球名称
- **点击星球卡片**：查看该星球的详细梦境列表（开发中）

#### 统计信息
- 梦境总数：所有记录的梦境数量
- 梦境星球：不同梦境类型的数量
- 虫洞连接：星球之间的连接数量

### 3. 梦境详情页面

#### 查看详情
- 梦境日期和时间
- 完整梦境内容
- 情绪分析可视化
- 梦境标签
- AI解梦（如果可用）

#### 操作功能
- **📤 分享**：分享梦境到微信
- **🗑️ 删除**：删除当前梦境

#### 相似梦境
- 页面底部显示相似的梦境
- 点击可以跳转到其他梦境详情

## 🐛 常见问题

### Q: 图标不显示？
A: 请参考"第一步：配置图标"添加图标文件，或临时注释 tabBar 配置。

### Q: 页面跳转失败？
A: 确保 `app.json` 中的页面路径正确，并且页面文件已创建。

### Q: 样式显示异常？
A: 检查是否使用了 Skyline 渲染引擎，某些 CSS 特性可能不支持。

### Q: AI分析没有反应？
A: 当前是模拟数据，真实AI功能需要接入后端API。
